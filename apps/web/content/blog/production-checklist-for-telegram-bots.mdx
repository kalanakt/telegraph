Before you scale traffic, lock in a short production checklist.

## Checklist

- Validate each flow before publish.
- Confirm bot token storage uses encryption.
- Set webhook URLs over HTTPS only.
- Verify queue backpressure and retry behavior.
- Review recent failed runs before launch windows.

## During launch

Use run history to spot bottlenecks by action type.

When a branch underperforms, isolate the condition and action pair first instead of changing multiple nodes at once.
